This is a summary of all the meetups I have attended in Singapore. 
Only a brief note is provided here for quick navigation, more details are in individual markdown files.
This section will be updated with latest meetups appearing at the top. 

#### [Humane Representations and Isochronic Singapore, Yin Shanyang](2017/20170925.md)
Toolchains for creating interactive data visualisations, public transport data, isochronic maps of singapore, mental models. 

#### [Machine Learning in Data Analytics, Ren Rouxu & Zane Lim](2017/20170916.md)
- NLP, deep learning myths, tensorboard 
- peaking into black-box models, PDP, lime, interpreting models, feature importance 

#### [Introduction to machine learning with Python, Vikrant Rathore](2017/20170912.md)
Basic stuff, still a good refresher of things to remember when running a machine learning model.

#### [Data Journalism, Jin Wu](2017/20170828.md)
Interesting Reuters articles, resources for creating interactive data visualisations.

#### [Data Science use cases at GO-JEK, Data Science Team](2017/20170824.md)
Use cases dicussion on driver ranking, recommender systems. Useful takeaways on working with real world data, deployment, model retuning, metrics and so on. 

#### [Working with social network data, Prasanta Bhattacharya](2017/20170815.md)
Introduction to network analysis, Tools in PyData stack for working with networks - networkx, igraph, graphtools, common pitfalls. 

#### [Practical data manipulation in R,  Dr. Pieter Fourie](2017/20170413.md)
`dplyr` package tutorial and use-cases in mobility simulation 

#### [Integrating Real-Time Data Streams with Spark and Kafka, Sengjoo Lim](2017/20170406.md)
Talk on [Talend](https://www.talend.com/), use case for streaming data analytics

#### [Pachyderm - Git for data science, Najib Ninaba](2017/20170316.md)  
On Pachyderm-Docker pipelines as an alternative to the Hadoop cluster for big data stacks.
